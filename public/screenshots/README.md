# Application Screenshots

This directory contains screenshots of the Clarsix application for display on the homepage and solutions page.

## Required Screenshots

Please add the following screenshot files to this directory:

1. **dashboard.png** - Dashboard view showing transaction statistics and recent activity
   - Recommended size: 1920x1200px (16:10 aspect ratio)
   
2. **transactions.png** - Transactions list page with filtering options
   - Recommended size: 1920x1200px (16:10 aspect ratio)
   
3. **transaction-details.png** - Transaction details page showing participants and timeline
   - Recommended size: 1920x1200px (16:10 aspect ratio)
   
4. **create-transaction.png** - Create transaction page with form
   - Recommended size: 1920x1200px (16:10 aspect ratio)
   
5. **payment.png** - Payment/initiate payment page
   - Recommended size: 1920x1200px (16:10 aspect ratio)

6. **join-transaction.png** - Join transaction page
   - Recommended size: 1920x1200px (16:10 aspect ratio)

## Screenshot Guidelines

- **Format**: PNG with transparency support
- **Resolution**: High-resolution (at least 1920px wide)
- **Aspect Ratio**: 16:10 or 16:9 for consistency
- **Content**: Make sure sensitive information is masked or replaced with demo data
- **Theme**: Take screenshots in both light and dark mode if showcasing theme switching

## How to Capture Screenshots

1. Navigate to each page in the application
2. Use browser dev tools to set viewport to 1920x1200
3. Capture full-page screenshots
4. Optimize images for web (compress without losing quality)
5. Save with the exact filenames listed above

## Naming Convention

File names should match exactly as specified above (lowercase, hyphenated).

Example:
- ✅ `dashboard.png`
- ❌ `Dashboard.png`
- ❌ `dashboard-view.png`
